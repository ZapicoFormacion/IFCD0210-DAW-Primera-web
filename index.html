HTML

Texto

En HTML se pueden añadir encabezados con diferente nivel de importancia y, por supuesto, párrafos que podrían ser formateados con negritas, cursivas, subrayados, e incluso tachados. En el caso de los encabezados se debería tener, por semántica, un único <h1> por página web, aunque se pueden utilizar cuantos sean necesarios de los restantes niveles, siempre teniendo en cuenta mantener un criterio lógico de anidación de niveles, tal y como se haría con el índice de un libro. 

            <h1>Encabezado de primer nivel</h1>
            <h2>Encabezado de segundo nivel</h2>
            <h3>Encabezado de tercer nivel</h3>
            <h4>Encabezado de cuarto nivel</h4>
            <h5>Encabezado de quinto nivel</h5>
            <h6>Encabezado de sexto nivel</h6>
            <p>Texto de un párrafo</p>
            <b>Texto en negrita</b>
            <i>Texto en cursiva</i>
            <u>Texto subrayado</u>
            <s>Texto tachado</s>

Listas

Es posible incluir listas ordenadas:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
            <ol>
                <li>Primer elemento</li>
                <li>Segundo elemento</li>
                <li>Tercer elemento</li>
            </ol>

Y también listas sin numeración:

  • Primer elemento
  • Segundo elemento
  • Tercer elemento
            <ul>
                <li>Primer elemento</li>
                <li>Segundo elemento</li>
                <li>Tercer elemento</li>
            </ul>

Textos pre-formateados

Por defecto, en los navegadores web, se muestra todo el texto HTML de continuo, independientemente de los saltos de línea que incluya el archivo html.
Es posible forzar un salto de línea empleando la etiqueta <br>, pero también es posible forzar y respetar, ya no sólo los saltos de línea, sino también los espacios ubicados al inicio de la línea para incluir contenidos que lo precisen, como por ejemplo código fuente. Además, en el caso de querer incluir código HTML será necesario reemplazar el símbolo < por lo que se conoce como su secuencia de escape &lt; para evitar que las etiquetas del código a mostrar sean interpretadas por el navegador.
En caso de necesidad, incluso se podía escapar cualquier carácter, como por ejemplo & que debería ser reemplazado por &#38; tal y como se ha hecho para poder mostrar este texto en el navegador que estás usando para leer estos contenidos. 

        Con diez cañones por banda,
        viento en popa a toda vela,
        no surca el mar, sino vuela,
        un velero bergantín.
        
            <pre>
                    Con diez cañones por banda,
                    viento en popa a toda vela,
                    no surca el mar, sino vuela,
                    un velero bergantín.
            </pre>
